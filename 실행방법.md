# 실행 방법 모음

## 공통 준비
- Docker Desktop 실행
- Node 20, Python 3.11 사용 (.nvmrc, .python-version 참고)
- 환경파일 필요 시: `cp .env.example .env` (루트)

## 도커 컴포즈로 전체 기동
```bash
docker compose -f infra/docker-compose.yml up -d
```
- 포트: 프론트 3000, API 8000, AI 9000, Oracle 1521/5500

## 백엔드(API) 수동 실행
```bash
cd apps/api
python -m venv venv
venv\Scripts\pip install --upgrade pip
venv\Scripts\pip install -r requirements.txt
set ORACLE_DSN=localhost:1521/FREEPDB1
set ORACLE_USER=lostfound
set ORACLE_PASSWORD=secret
set JWT_SECRET=change-me
venv\Scripts\uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## 프론트엔드 실행 (Turbopack 제외)
```bash
cd apps/frontend
npm install --ignore-scripts
set NODE_ENV=development
npx next dev -H 0.0.0.0 -p 3000
```

## AI 서비스 실행 (옵션)
```bash
cd services/ai
python -m venv venv
.\venv\Scripts\python.exe -m pip install --upgrade pip   # 생략 가능
.\venv\Scripts\python.exe -m pip install -r requirements.txt
set LLM_BASE_URL=http://localhost:1234/v1   # 필요 시 설정
.\venv\Scripts\uvicorn app.main:app --host 0.0.0.0 --port 9000
```
